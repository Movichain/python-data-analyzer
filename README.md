# Python Data Analyzer

A Python script that takes raw CSV data, uses Pandas and NumPy 
to clean, group, filter and calculate trends, and generates 
a structured summary report.

## Skills Demonstrated
- Pandas DataFrame operations (groupby, filter, sort)
- NumPy array operations and statistical calculations
- Raw data cleaning and validation
- Data aggregation and trend analysis
- Structured report generation

## Technologies
- Python 3
- pandas
- numpy

## How to Run
```bash
pip install pandas numpy
python analyzer.py
```
